# Site Institucional

Site institucional moderno construído com TypeScript, Vite e Tailwind CSS.

## Sobre

Template de site institucional com tooling moderno: TypeScript para tipagem, Vite para build rápido e Tailwind CSS para estilização. Estrutura pronta para customização e deploy.

## Stack

| Tecnologia | Uso |
|-----------|-----|
| TypeScript | Linguagem principal |
| Vite | Build tool e dev server |
| Tailwind CSS | Framework de estilização |
| PostCSS | Processamento de CSS |

## Como rodar

```bash
npm install
npm run dev      # Dev server com hot reload
npm run build    # Build de produção
npm run preview  # Preview do build
```

## Estrutura

```
src/            # Código TypeScript
public/         # Assets estáticos
vite.config.ts  # Configuração Vite
tsconfig.json   # Configuração TypeScript
tailwind.config # Configuração Tailwind
postcss.config  # Configuração PostCSS
```
